### Directory Structure

- **history-substring**
- **syntax-highlight**

### Custom functions
- **upload** : upload a file to server
- **pastebin** : upload a file to pastebin
- **server** : Runs a simple HTTP server
- **gip** : Get the global IP address
- **lip** : Get local IP
- **proxy** : set unset proxy
- **cd magic** : go to root git directory or ~
